# ­ЪћЌ URL рЃЊрЃћрЃЎрЃЮрЃЊрЃћрЃарЃў / рЃћрЃюрЃЎрЃЮрЃЊрЃћрЃарЃў

## ­ЪЊќ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
рЃћрЃА рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЋрЃћрЃЉ-рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃњрЃљрЃюрЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЮрЃЌ **URL-рЃўрЃА рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ** рЃЊрЃљ **рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ**.  
рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ **HTML, JavaScript** рЃЊрЃљ **Tailwind CSS**-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃўрЃА рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљрЃА рЃДрЃЋрЃћрЃџрЃљ рЃбрЃўрЃърЃўрЃА рЃћрЃЎрЃарЃљрЃюрЃЌрЃљрЃю рЃЊрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃАрЃЌрЃљрЃю.

---

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў
- **URL рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ** РђЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃА URL-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў (`encodeURIComponent`).
- **URL рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ** РђЊ рЃљрЃдрЃљрЃЊрЃњрЃћрЃюрЃА рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃБрЃџ URL-рЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў (`decodeURIComponent`).
- **рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ** РђЊ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃњрЃљрЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃЮрЃЌ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃљрЃю рЃњрЃљрЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў.
- **рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ** РђЊ рЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃљрЃю рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЋрЃћрЃџрЃА.
- **рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃЊрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃљ** РђЊ рЃфрЃЋрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА.
- **рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў** РђЊ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃЊрЃћрЃЉрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃљрЃќрЃћ (рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў, рЃбрЃљрЃЉрЃџрЃћрЃбрЃў, рЃЊрЃћрЃАрЃЎрЃбрЃЮрЃърЃў).
- **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў** РђЊ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў Tailwind CSS-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

---

## ­Ъџђ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
- **рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃЋрЃћрЃџрЃў** РђЊ рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ URL рЃљрЃю рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ рЃљрЃю рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ рЃњрЃАрЃБрЃарЃЌ.  
- **рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў**:
  - РќХ№ИЈ **рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ** РђЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃБрЃџ URL-рЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.
  - РќХ№ИЈ **рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ** РђЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃА URL-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.
  - ­Ъћё **рЃерЃћрЃфрЃЋрЃџрЃљ** РђЊ рЃфрЃЋрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА.
  - ­ЪЊІ **рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ** РђЊ рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃљрЃю рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃА.
  - ­ЪД╣ **рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ** РђЊ рЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЋрЃћрЃџрЃА.
- **рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЋрЃћрЃџрЃў** РђЊ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџ рЃерЃћрЃЊрЃћрЃњрЃА.

---

## ­ЪЏа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў
- **HTML** РђЊ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ  
- **JavaScript** РђЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў (рЃћрЃюрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ, рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ, рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ, рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ)  
- **Tailwind CSS** РђЊ рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў  

---

## ­Ъњ╗ рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
- рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃћрЃЉ-рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў: Chrome, Firefox, Safari, Edge  
- рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб-рЃЎрЃљрЃЋрЃерЃўрЃарЃў Tailwind CSS-рЃўрЃА CDN-рЃўрЃА рЃЕрЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ  

---

## ­ЪЊё рЃџрЃўрЃфрЃћрЃюрЃќрЃўрЃљ
